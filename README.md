## &sect;问题抽象

* 目标：获得真实分数函数：`f(**S**)=**s**`
* 其中：
  * `**S**`：
    * 原始评分矩阵，为m*n的矩阵；
    * m为评分人数，n为被评分人数；
    * 矩阵中没有缺失数据，即数据已经经过预处理；
  * `**s**`：最终分数向量，为n维向量，代表每个人最终的分数。
* 影响准确性的因素：
  1. 每个人的评分标准不同
  2. 每个人对自己和别人的评分标准不同

## &sect;处理算法

1. 消除个人评分标准导致的误差：对每个评分人的评分进行标准化，得到每个被评分人的偏差值，再对每个被评分人的偏差值进行平均。
2. 调整整体分数：将每个人的偏差值映射至与原数据具有相同的均值和离散程度的空间中，得到最后的分数。
